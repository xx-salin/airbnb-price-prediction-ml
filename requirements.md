# Requirements
 
## R Version
R 4.0 or higher
 
## R Packages
The script will automatically install missing packages on first run. The following are required:
 
- `tidyverse`
- `skimr`
- `DataExplorer`
- `GGally`
- `factoextra`
- `knitr`
- `readr`
- `dplyr`
- `caret`
- `tidymodels`
- `randomForest`
- `keras`
- `nnet`
- `kableExtra`
- `glmnet`
## Data
Place `LONlistings.csv` in the `data/` folder before running.  
Dataset source: [Inside Airbnb](http://insideairbnb.com/get-the-data/) — London, December 2024.
 
## Running the Script
Open `349Analysis.R` in RStudio and run top to bottom. All outputs (plots, tables) are generated inline.
